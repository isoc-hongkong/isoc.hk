The program will also incorporate hands-on exercises for practical deployment. **WHO SHOULD JOIN?**
          * Network/ System Engineers, IT/ Product/ Operation Managers, System Administrators
          * Those who would like to obtain IPv6 qualification
  **COURSE DESCRIPTION** The IPv6 Certification and Training Program (2 days) includes: 
          * Introduction, history and the Need for IPv6
          * Basic IPv6 Concepts
          * IPv6 Address Structure
          * IPv6 Nomenclature
          * IPv6 Address Planning Techniques
          * Basic IPv6 Interface Configuration
          * Building IPv6 Tunnels where necessary
          * Initial static routing
          * Deploying IPv6 DNS
          * Deploying IPv6 DNS Glue Records
          * Deploying IPv6 Web Servers
          * Deploying IPv6 Email Services (SMTP/Sendmail, IMAP/Dovecot)
          * Hands-on Exercises for Practical Deployment
          * Preparation for HE's worldwide online IPv6 certification program
